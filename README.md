# SafeMessage

**Mensagens encriptadas para comunicação clínica segura**

> Live: [psynoord.com/pt/safemessage](https://psynoord.com/pt/safemessage)

Ferramenta de envio de mensagens confidenciais com encriptação AES-256-GCM, desenvolvida para psicólogos e profissionais de saúde nos Países Baixos.

## Stack
- **Backend:** Node.js · Express
- **Encriptação:** AES-256-GCM (military-grade)
- **Storage:** Auto-delete após leitura

## Funcionalidades
- Encriptação AES-256-GCM no servidor
- Código de 6 dígitos para acesso
- Auto-eliminação após leitura
- Suporte a ficheiros até 2MB
- Sem dados pessoais armazenados
